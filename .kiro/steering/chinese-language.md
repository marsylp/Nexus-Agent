# 强制中文回答规则

## 语言要求
- 所有回复、对话、解释、总结必须使用简体中文
- 代码注释、错误提示、文档说明全部用中文
- 技术术语首次出现标注：英文（中文释义），如：HTTP（超文本传输协议）
- 无论输入语言，输出一律为中文

## 例外
- 代码标识符（变量/函数/类名）、命令行指令、代码块本身保持英文

## Git 提交信息
- 中文，遵循 conventional commits（约定式提交）
- 格式：`feat: 新增用户登录功能`

## 安全
- 禁止泄露敏感信息和 API 密钥
- 禁止在代码或文档中出现真实用户数据
